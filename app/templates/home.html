{% trans %}Editing with Live Preview and SCSS{% endtrans %}

{% trans %} With {{ appname }}, you get an in-browser, feature-rich, syntax-highlighting CSS editor. You can see your changes instantly with the side-by-side preview pane, so there's no need for the traditional back-and-forth between your editor and refreshing a browser to see your changes. {% trans %}Easy Setup, Publishing and Hosting{% endtrans %}

{% trans %} Setting up {{ appname }} on your website is as easy as adding two teeny-tiny tags (a link tag and a script tag) to the end of your site's <head> element. No pushing to a repository, updating a remote file system, or restarting your web app required. Just edit, preview, publish, and move on.
